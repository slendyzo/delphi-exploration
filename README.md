# Shill Feature Demo

A demo of a feature allowing users to submit and view "shill" ideas for various assets.

## Features
- Submit new trade ideas (symbol, network, thesis, position).
- View a list of submitted ideas.
- Click on an idea to see details and comments.
- Leaderboard to view top-performing ideas (mock data).
- Basic filtering on the leaderboard.

## Built With
- HTML
- CSS
- JavaScript (Vanilla)
